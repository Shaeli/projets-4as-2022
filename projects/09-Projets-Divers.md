# Projet Amélioration de la salle

## Description

Dépendra du statut de la salle 4AS

## Possibilités (liste non-exhaustive): 

* Honeypot
* VPN pour acceder a l’infrastructure de la salle
* Radius + 802.1X pour le Wifi
* Firewall / Proxy
* GPO / Windows hardening
